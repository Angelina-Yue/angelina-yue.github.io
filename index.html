像游戏主页面一样：Start / Rule / Gallery

✨ Rare Glow：A=金光 / B=紫光 / C=粉光（无屏幕震动）

· 点击「START」进入抽赏界面
· 抽一次：盒子摇晃 → 开盖 → 奖品慢慢变大弹出
· 奖池会扣减，抽完可重置
· Gallery 可以放你想展示的几张图片

快速说明

想放图片：把图片上传到仓库的 assets/ 文件夹，然后在 Gallery 里写 assets/xxx.jpg 就能显示。
提示：如果手机缓存没更新，用 ?v=999 刷新即可。

RULE · 玩法说明

玩法：点击「抽一次」进行抽赏。
稀有光效：A=金光 / B=紫光 / C=粉光，并有庆祝粒子。
奖池：抽到的奖项会从剩余数量中扣减，抽完可点「重置奖池」重新开始。

修改奖池数量：在代码里找到 initialPrizes，改每个奖项的 count。

GALLERY · 图片展示

怎么放图片：
1）在 GitHub 仓库里点 Add file → Upload files
2）建议新建文件夹 assets（没有就先创建），把图片传进去
3）图片路径示例：assets/1.jpg、assets/kuromi1.png
4）然后在代码里找到 galleryImages 数组，把路径填进去即可
